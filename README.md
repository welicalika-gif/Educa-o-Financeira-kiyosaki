# Educa-o-Fina📊 Projeto: Educação Financeira com Robert Kiyosaki
📌 Descrição

Este projeto apresenta um mini guia de educação financeira baseado nos ensinamentos de Robert Kiyosaki, autor do livro Rich Dad Poor Dad.

O objetivo é explicar conceitos fundamentais sobre dinheiro, investimentos e mentalidade financeira.

🧠 Conceitos principais
1️⃣ Ativos e Passivos

Segundo Kiyosaki:

Ativos → colocam dinheiro no seu bolso

Passivos → tiram dinheiro do seu bolso

Exemplos:

Ativos	Passivos
Investimentos	Dívidas
Imóveis que geram renda	Carros financiados
Negócios	Cartões de crédito
2️⃣ Educação Financeira

A educação financeira é essencial para que as pessoas aprendam a:

administrar dinheiro

investir

evitar dívidas

construir riqueza

3️⃣ Mentalidade Financeira

Kiyosaki defende que a escola tradicional ensina a trabalhar por dinheiro, enquanto a educação financeira ensina o dinheiro a trabalhar para você.
